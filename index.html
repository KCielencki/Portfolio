Flight Price Prediction Tool
The airline price predictor is an interactive tool created to predict airline prices for the next available calender day given an origin and a destination airport. This is complete with visualizations for the end-user to gain meanningful insight into ticket price trends based on seasonality and origin/destination airports. This tool is based on Python machine learning model using data from Bureau of Transportation Statistics.
Tools Used: Python, JavaScript, HTML, Tableau, teamwork

New York City Shootings Visualizations
I analyzed data from the City of New York Open Data (“Open Data”) and presented a number of dynamic, user-friendly visualizations. The data was retrieved from the Open Data API, visualized with JavaScript libraries (D3 and Plotly) and hosted on an HTML webpage with dynamic dashboards. The visualizations are laid out in a way to provide the end-user with a meaningful story of gun violence broken out by neighborhood, age and sex of victim as well as year. This allow the end-user to observe long-term trends of gun violence in New York City.
Tools Used: JavaScript, HTML

COVID-19 Analysis
Investigating the trends and factors causing the spread of COVID-19 in the U.S. This was created using data from John Hopkins and US Census to explore areas, such as, but not limited to: population, density, and demographics (data covers January 2020 through April 2020). This is presented as a powerpoint deck using Python visualizations to lay out a story illustrating the spread and impact of the COVID-19 pandemic.
